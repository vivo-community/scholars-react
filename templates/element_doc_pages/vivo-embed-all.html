Embeddable Components

These are web components that can be imported within any site to display lists of data. They are in the assets directory, unhashed when compiled, so available as straight imports e.g.

<%= raw(` <script type="text/javascript" src="https://vivo-scholars-scholars.cloud.duke.edu/assets/vivo-embed-all.js"></script> `) %>

Publication List

Set the endpoint and person-id attributes

<%= raw(` <vivo-embedded-publication-list endpoint="https://scholars-discovery-scholars.cloud.duke.edu/graphql" person-id="n2363"> </vivo-embedded-publication-list> `) %>

Grant List

Set the endpoint and person-id attributes

<%= raw(` <vivo-embedded-grant-list endpoint="https://scholars-discovery-scholars.cloud.duke.edu/graphql" person-id="n3323"> </vivo-embedded-grant-list> `) %>

Course List

Set the endpoint and person-id attributes

<%= raw(` <vivo-embedded-course-list endpoint="https://scholars-discovery-scholars.cloud.duke.edu/graphql" person-id="n3323"> </vivo-embedded-course-list> `) %>

People List (of Organization)

Set the endpoint, type and organization attributes

<%= raw(` <vivo-embedded-organization-people-list endpoint="https://scholars-discovery-scholars.cloud.duke.edu/graphql" type="schools" organization="School of Arts and Sciences"> </vivo-embedded-organization-people-list> `) %>
